## Overview

The Hello Future Hackathon is a three-part innovation series on Hedera:
- **Origins** (Summer 2025) — $150,000 prize pool
- **Ascension** (November 2025) — $150,000 prize pool
- **Apex** (Feb–Mar 2026) — $250,000 prize pool ← **current**

**Total trilogy prize pool: $550,000**

No prior participation required — new builders are welcome.

Organizers: **AngelHack** + **Hashgraph (Hedera)**

## Submission Requirements

All of the following are mandatory:

1. **GitHub repository** — code + README
2. **Project description** — max 100 words
3. **Tech stack** listing
4. **Pitch deck** — PDF format
5. **Demo video** — max 5 minutes, YouTube link required
6. **Live demo link**

**Rules:**
- 1 main track submission per team
- 1 bounty submission per team
- Maximum: one main track + one bounty
